# 10 民泊事業

ルート: `/minpaku`
仮想ブランド名: **「やまどり庵」** — 京都西陣の町家を改装した1日1組限定の宿

## コンセプト

「静かに、町家に、暮らすように泊まる」。インバウンド向けに見えすぎる派手な民泊ではなく、**国内シニア・夫婦・記念日層**を意識した上質な町家ステイ。

## フォント

- 見出し: **Shippori Mincho**（縦書きも使う）
- 欧文: **Cormorant Garamond**
- 本文: Noto Serif JP

## レイアウト方針

- 余白を大胆に取り、視線を画像に集中させる
- 縦書き見出しを1〜2か所配置
- 季節感（春の桜／夏の風鈴／秋の紅葉／冬の雪）が伝わる画像を中心に配置

## 主要アニメーション

1. **ヒーロー**: 障子が左右に開くマスクアニメ → 中庭写真が現れ、ゆっくりズームイン（GSAP）
2. **縦書きキャッチ**: 1文字ずつ墨が滲むようにフェードイン（SplitText + filter blur）
3. **客室セクション**: 横スクロールで2部屋を見せる（GSAP ScrollTrigger）
4. **季節ギャラリー**: 4枚の画像が視差で重なる（react-spring/parallax）
5. **CTA**: 「お問い合わせ」ボタンに筆ストロークのSVGアニメ
