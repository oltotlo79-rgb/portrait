# 90 実績一覧トップ（ポートフォリオインデックス）

ルート: `/`

## 役割

クラウドワークス提案文に貼る URL の本命。**自身の紹介は最小限**、サンプルへの導線に特化したギャラリー。

## ページ構成

```
HERO         "Web Production / 12 Industries"
             短いキャッチ＋スクロールヒント

WORKS GRID   12業種カード（3×4 / モバイル1列）
             カード = 業種名・キャッチ・サムネ・配色

CONTACT      お問い合わせフォーム（モック）
```

トップ1ページ完結（深い情報構造は持たない）。

## ヒーローのコピー案

> Web Production
> for any industry.
>
> 業種ごとの世界観で、選ばれるホームページを。
>
> 12 sample sites — scroll to explore ↓

## カードに表示する情報

| 項目 | 内容 |
|---|---|
| 業種ラベル | 「Restaurant 割烹」など英＋和 |
| キャッチ | 各サイト計画のヒーローコピーを30字程度で要約 |
| キーカラー | 業種カラー2色のグラデ |
| サムネ画像 | ヒーロー画像を 4:5 にトリミング |
| ステータスバッジ | "LIVE" / "WIP"（Phaseで切替） |
| 矢印 | ホバーで右に伸びる |

## アニメーション

- カードはGSAP ScrollTriggerで段階的に登場（`stagger: 0.08`）
- ホバーでサムネがスケール1.05＋業種テーマ色オーバーレイ
- HEROタイトルは GSAP SplitText で1文字ずつ降下、続けて Lenis のスクロール演出
- ページ全体に上品なノイズオーバーレイ（`NoiseOverlay`）

## カラー

このページは**ニュートラル**にする：
- 背景 `#0E0E0E`
- 文字 `#F6F6F2`
- ホバー時のみ各業種カラーが浮かぶ → 12業種の世界観の違いを際立たせる
