## レイアウト方針

- 手書き要素・SVGステッカー・ふきだしを散らす
- 角丸を大きめ（24〜32px）に
- 写真と手書きイラストの重ね合わせ

## 主要アニメーション

1. **ヒーロー**: 手書き風タイトルが一画ずつ描かれる（GSAP DrawSVG）。背景に紙吹雪Lottie
2. **プログラムカード**: ホバーでカードがぴょこんと跳ねる（spring stiffness高め）
3. **保護者の声**: 吹き出しが横にぽよんと並んで流れるMarquee
4. **クレヨンライン**: スクロールに合わせて手書き線がページを縦断
5. **CTA**: 「体験してみる」ボタンが揺れる＋★がランダムに散る
