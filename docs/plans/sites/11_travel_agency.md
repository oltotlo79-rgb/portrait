# 11 旅行斡旋業

ルート: `/travel`
仮想ブランド名: **「HORIZON Travel」** — 大人のためのオーダーメイド旅行代理店

## コンセプト

パッケージ旅行ではなく、「相談から作る、世界に1つの旅」。ヨーロッパ・北欧・中東・南米など、**少し非日常な目的地**にフォーカスし、写真の力で「行きたい」を喚起する。

## レイアウト方針

- 全幅写真でセクション仕切り、目的地ごとに1スクリーン
- 余白少なめ、画像の引力で読ませる
- 旅程・コンセプトは「箇条書きより、語り」を意識（コラム調）

## 主要アニメーション

1. **ヒーロー**: 巨大な写真スライダーが3カット切り替わりながらピン留め＋テキストMask Reveal（GSAP）
2. **目的地セクション**: **水平スクロール**で6地域を進む。各地域でテキストが視差で動く
3. **数字カウンター**: 「世界◯カ国」「累計◯件」がスクロールで数字アップ
4. **コンシェルジュ**: ホバーで写真がモノクロ→カラーに切り替わり、サインのSVGがストローク
5. **CTA**: マグネティックボタン＋背景がスクロール速度で歪む
