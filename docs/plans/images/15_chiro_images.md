# 15 整体「ことう整体院」画像生成指示

対応サイト計画: [`sites/15_chiropractic.md`](../sites/15_chiropractic.md)
配置先: `public/images/chiro/`

## トーン

- 慢性的な肩こり・腰痛のための整体院
- 清潔、安心、白・青緑・淡水
- 派手な動きはなく、信頼の伝わる写真
- 院長の人柄が伝わるトーン

## ライティング

- 高めの自然光、影は柔らかく
- ハイキー、コントラスト弱め
- 病院っぽくならない（過剰に白すぎない）

## 入れない要素

- 白衣の医師っぽい強い写真（病院感）
- 痛がる患者の顔
- 派手な医療器具・大型機器
- ガッツポーズ等の元気アピール

## SVG装飾

- `deco-wave.svg`: ゆるい波線（青緑）
- `deco-checkmark.svg`: チェックマーク（青緑、太めライン）
