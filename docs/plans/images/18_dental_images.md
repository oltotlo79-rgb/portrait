# 18 歯科「白水歯科クリニック」画像生成指示

対応サイト計画: [`sites/18_dental_clinic.md`](../sites/18_dental_clinic.md)
配置先: `public/images/dental/`

## トーン

- 痛みの少ない予防型歯科
- 清潔・透明感・余白
- 親子で来られる安心感、白基調＋ミントブルー
- 怖そうな器具は写さない

## ライティング

- ハイキー、自然光
- 真っ白すぎないクリーム寄り
- ミント系のアクセントが映える色温度

## 入れない要素

- 鋭利な器具・血・赤い口腔写真
- 子どもが泣いている写真
- ストック感の強い白衣集合
- 派手なロゴやイラスト
