# 23 ベーカリー「麦と灯（むぎとひ）」画像生成指示

## トーン

- 朝5時に火を入れ、6時から焼き始める
- 街の小さなブーランジェリー、夫婦経営の温度感
- 粉と水と塩、ただそれだけの時間
- 焼き色・小麦・キャラメリゼ・蜂蜜のゴールド

## カラーパレット

- 生成り背景 `#FBF6ED`
- 焼き色 `#6B4423`
- ダーク `#3A2A18`
- ゴールド `#D4A647`

## ライティング

- 早朝・午前の自然光、窓越しの黄金光
- 粉が舞う光線（ティンダル現象）を1〜2枚
- スタジオライト不使用、家庭的・温度のある質感
- 影は柔らかく、ハイライトは強すぎない
