# 02 旅行「HORIZON Travel」画像再生成プロンプト

対応サイト: `/travel`
配置先: `public/images/travel/`
ブランド: HORIZON Travel — 大人のためのオーダーメイド旅行
カラーパレット: メイン `#0F4C81`（深ブルー）／アクセント `#F4B400`（陽光）／背景 `#F7F4EE`

## 共通トーン

- **写真**: シネマティック・トラベルフォトジャーナリズム（Nat Geo Traveler）
- **フィルム参照**: Kodak Portra 400 / Cinestill 50D
- **レンズ**: 24mm（広角・風景）／50mm（標準）／85mm（人物）
- **時間帯**: マジックアワー、朝靄、青の時間
- **雰囲気**: 開放感、地平線、空気遠近、シネマトーン
- **避ける**: 観光地の人混み、自撮り風、SNS加工、ストック家族写真
