# 00 全体概要

## ゴール

クラウドワークスでHP制作案件を受注するため、**「業種特化＋動きのあるモダンサイト」**を複数本そろえた実績ポートフォリオを構築する。

## ターゲットクライアント像（CW受注先）

- 個人事業主〜従業員30名規模の中小企業
- 「テンプレ感のないHPがほしい」「動きや演出にこだわりたい」と感じている層
- 予算感: 10〜50万円のHP制作案件
- 業種は問わないが、業種特化の事例を見せることで「自分の業界に強そう」という信頼を獲得

## 成功指標

1. クラウドワークス提案文に貼るURLを1本（実績一覧トップ）持っている状態
2. 業種ごとの個別URL（`/restaurant`, `/salon` など）も提案先に応じて差し替え可能
3. 全サイトでLighthouse Performance 90+、視覚的なリッチさを担保
4. リッチアニメーション（スクロール演出・ホバーマイクロインタラクション・ローディング演出）を全サイトに搭載

## スコープ

### やる
- 12業種のサンプルサイト
- 実績一覧トップページ（自身の紹介は最小限、サンプルへの導線に特化）
- 業種ごとに個性ある配色・タイポ・モーション
- Cloudflare Pages静的書き出しによる公開
- Codexで生成する画像の指示書一式

### やらない
- 多言語対応（日本語のみ）
- お問い合わせフォームの実送信（UIモックのみ）
- 管理画面・CMS
- 個別ドメイン取得（`*.pages.dev` サブドメインで運用）
- 制作者個人を売り込む紹介ページ（一覧ページのみで足りる）

## 制約

- 静的書き出し前提のため、Next.jsのSSR/ISR/サーバーアクションは使用しない
- 画像は最終的に `public/images/<site>/` 配下に配置し、`<Image unoptimized />` で参照
- アニメーションは過剰になるとUX/パフォーマンスを損なうため、各サイト「ヒーロー・主要セクション・実績/メニュー」の3〜4箇所に重点配置
